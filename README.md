# Personal Portfolio Starter

A minimalist personal portfolio starter built with Next.js (App Router), inspired by the visual style you shared.

## Requirements

- Node.js `>= 20.9.0` (Next.js 16 requirement)

## Run locally

```bash
nvm use
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Deploy to Vercel

1. Push this folder to GitHub.
2. Import the repo in Vercel.
3. Framework preset: `Next.js`.
4. Build command: `next build` (default).
5. Output: auto-detected by Vercel.

## Language switch

- English: `/?lang=en`
- Chinese: `/?lang=zh`

## Quick customization

- Main page structure and content: `app/page.tsx`
- Global style (colors, spacing, animation): `app/globals.css`
- Site metadata and fonts: `app/layout.tsx`

## Notes

- Replace placeholder links (`#`, `https://x.com`, `https://cal.com`) before production.
